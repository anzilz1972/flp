# 文字冒险游戏伪代码框架

## 主要知识点总结

1. **match语句** - 评估主题值并与模式匹配，类似switch但更强大
2. **字面量模式** - 匹配特定值（如 "quit", 404）
3. **捕获模式** - 绑定变量从主题值
4. **序列模式** - 解包列表/元组
5. **通配符(_)** - 始终匹配但不绑定
6. **或模式(|)** - 多个模式对应同一结果
7. **as模式** - 匹配模式同时绑定变量
8. **guard守卫** - 模式后的if条件
9. **映射模式** - 匹配字典键值
10. **类模式** - 匹配对象类型和属性

## 游戏框架

### 主循环

```
主程序入口
    初始化游戏状态
    循环直到游戏结束:
        显示当前房间描述
        获取玩家输入
        解析命令并执行:
            match command.split():
                case ["quit"]:
                    退出游戏
                case ["look"]:
                    显示当前房间
                case ["get", obj]:
                    获取物品
                case ["drop", *objects]:
                    丢弃多个物品
                case ["go", direction] if direction in 当前可通行方向:
                    移动到相邻房间
                case ["go", _]:
                    显示无法前往提示
                case ["inventory"]:
                    显示背包物品
                case _:
                    显示无法识别命令
```

### 命令处理模式

```
单命令模式:
    case ["look"] | ["l"]:
    case ["quit"] | ["exit"]:

双命令模式:
    case ["get", obj]:
    case ["drop", obj]:
    case ["go", direction]:
    case ["attack", target]:

多值命令模式:
    case ["drop", *items]:
    case ["buy", *items]:

别名/或模式:
    case ["north"] | ["n"] | ["go", "north"]:
    case ["get", obj] | ["pick", "up", obj] | ["pick", obj, "up"]:

带守卫的命令:
    case ["go", direction] if direction in exits:
        执行移动
    case ["go", _]:
        显示错误
```

### 事件处理模式（UI扩展）

```
match event.get():
    case Click(position=(x, y), button=Button.LEFT):
        处理左键点击
    case KeyPress(key_name="up arrow"):
        向上移动
    case KeyPress(key_name="Q"):
        退出游戏
    case Quit():
        清理并退出
    case _:
        忽略其他事件
```

### 网络通信模式（JSON映射）

```
match action:
    case {"text": str(msg), "color": str(c)}:
        显示带颜色的文本
    case {"sleep": float(duration)}:
        等待指定时间
    case {"sound": str(url), "format": "ogg"}:
        播放音频
    case {"sound": _, "format": _}:
        警告不支持的格式
```

## 完整游戏框架

```
类 Game:
    属性:
        player (玩家对象)
        rooms (房间字典)
        current_room (当前位置)
        running (游戏循环标志)

    方法:
        开始():
            初始化游戏
            运行主循环

        获取输入():
            返回 input() 拆分结果

        处理命令(command):
            match command.split():
                # 退出
                case ["quit"] | ["exit"]:
                    self.running = False
                    return "再见!"

                # 查看
                case ["look"] | ["l"]:
                    return self.current_room.describe()

                # 移动
                case ["north"] | ["n"] | ["go", "north"]:
                    return self.移动("north")
                case ["south"] | ["s"] | ["go", "south"]:
                    return self.移动("south")
                case ["east"] | ["e"] | ["go", "east"]:
                    return self.移动("east")
                case ["west"] | ["w"] | ["go", "west"]:
                    return self.移动("west")
                case ["go", direction] if direction in self.current_room.exits:
                    return self.移动(direction)
                case ["go", _]:
                    return "无法前往该方向"

                # 物品操作
                case ["get", obj] | ["pick", "up", obj] | ["pick", obj, "up"]:
                    return self.player.获取(obj, self.current_room)
                case ["drop", *objects]:
                    results = []
                    for obj in objects:
                        results.append(self.player.丢弃(obj, self.current_room))
                    return "\n".join(results)
                case ["inventory"] | ["inv"] | ["i"]:
                    return self.player.显示背包()

                # 攻击
                case ["attack", target] | ["kill", target]:
                    return self.player.攻击(target, self.current_room)

                # 购买(商店)
                case ["buy", *items]:
                    return self.商店.购买(items, self.player)

                # 通配符
                case _:
                    return f"无法理解命令: {command}"

        移动(direction):
            if direction in self.current_room.exits:
                self.current_room = self.current_room.exits[direction]
                return self.current_room.describe()
            return "无法前往该方向"

        主循环():
            self.running = True
            while self.running:
                显示当前状态
                获取玩家输入
                处理命令
                显示结果

# 启动游戏
if __name__ == "__main__":
    game = Game()
    game.开始()
```

## 知识点应用对照表

| 知识点 | 游戏中的应用 |
|--------|-------------|
| match语句 | 命令解析主结构 |
| 字面量模式 | 匹配 "quit", "look" 等命令 |
| 捕获模式 | 绑定 action, obj, direction 变量 |
| 序列模式 | 解包 command.split() 列表 |
| 通配符(_) | 未知命令和多余参数处理 |
| 或模式(\|) | 命令别名 (n/north/go north) |
| as模式 | 捕获子模式值 |
| guard | 方向检查、物品存在检查 |
| 映射模式 | JSON消息解析 |
| 类模式 | UI事件处理 |
